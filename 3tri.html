Funções Modulares

Módulos

Primeiramente vou explicar o que é um módulo, ja que não faria nenhum sentido eu explicar o resto sem dizer o básico. A idéia dos módulos é meio como medir distâncias no mundo dos números. Se você tem A e B, módulo A - B é como calcular a distância entre A e B, sem se preocupar se é um é maior que o outro. Por exemplo, se A = 7 e B = 4, então | A - B | = 3, a distância entre 7 e 4

Equação Modular

Agora, se você tem uma equação modular, é como resolver um mistério de distância. Se temos | X - 3 | = 5, queremos descobrir que número X está a uma distância de 5 unidades do número 3, pode ser que X = 8 ou X = -2, porque ambos estão a 5 unidades de distância do 3, Então, resolver uma equação modular é como descobrir quem cometeu o "crime" da distância.

Função Modular

Imagina que as funções modulares são como despertadores matemáticos. Se você define uma função como f(x) = |x|, ela retorna sempre o valor absoluto de x. Ou seja, se x é negativo, a função modular "acorda" o valor positivo correspondente. Funciona como se estivéssemos interessados apenas na distância do número até zero, ignorando a direção

PA - Progressão Aritmética

Progressão Aritmética (PA):

Pensa em uma PA como uma sequência matemática que gosta de seguir uma ordem específica. Se você tem o primeiro termo a e uma razão r, cada termo subsequente é como dar um passo consistente. Por exemplo, se o primeiro termo é 3 e a razão é 2, a sequência seria 3, 5, 7, 9, e assim por diante, sempre adicionando 2.

Termo Geral:

O termo geral de uma PA é basicamente uma fórmula mágica que te dá qualquer termo que você quiser. Para encontrar o enésimo termo (a_n), você usa a fórmula a_n = a + (n-1) * r. É como se cada termo tivesse seu endereço próprio na "casa" da sequência.

Soma dos Termos:

A soma dos termos de uma PA é como somar os valores de cada casa da sequência. Se você quiser somar os primeiros n termos, a fórmula é S_n = n/2 * [2a + (n-1)r]. É como se fosse uma festa, e a fórmula te dá o total dos presentes que cada termo traz.

Progressão Geométrica (PG):

Uma PG é uma sequência de números onde cada termo, a partir do segundo, é obtido multiplicando o termo anterior por uma constante chamada razão (r). A fórmula do enésimo termo (a_n) em uma PG é dada por a_n = a * r^(n-1), onde a é o primeiro termo e n é o número do termo desejado.

Razão:

A razão (r) em uma PG representa a constante pela qual cada termo é multiplicado para obter o próximo termo. Se r > 1, a PG é crescente; se 0 < r < 1, é decrescente.

Soma dos Termos:

A soma dos primeiros n termos de uma PG, denotada por S_n, é dada pela fórmula S_n = (a * (r^n - 1)) / (r - 1) para r ≠ 1. Quando r = 1, a fórmula se simplifica para S_n = n * a.

Matemática Financeira

Juros Simples:

Se você emprestou ou investiu dinheiro, juros simples são como um aluguel pelo uso desse dinheiro. Com um principal (valor inicial), taxa de juros e tempo, os juros são proporcionais ao principal e ao tempo. É como pagar ou receber uma quantia fixa por um período.

Montante:

O montante (M) é o saldo total após considerar juros. Se começou com um principal (P), os juros simples (J) são P * taxa * tempo, e o montante final é a soma do principal e dos juros: M = P + J. É o principal mais a taxa de "aluguel" multiplicada pelo tempo.

Juros Compostos:

Juros compostos são como o dinheiro gerando mais dinheiro. A fórmula M = P * (1 + taxa)^tempo expressa esse crescimento exponencial, adicionando "andares" na escada dos juros a cada período.
